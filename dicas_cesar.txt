- Não usar parágrafos tão curtos

- Discutir mais profundamente o problema de pesquisa. Usar referências para apresentar o problema.
  Problema de pesquisa deve mostrar que há uma discussão acadêmica, com autores comentando na literatura.
  Não deve ser apenas uma lista de vantagens e desvantagens.
  Qual é a necessidade?  Por que é necessário? Quais são as dificuldades?
  O que estimula a buscar a solução proposta?
  As soluções existentes são necessárias para resolver o problema?
  Não tentar falar do problema com a solução.
  Mostrar que há uma necessidade real.
